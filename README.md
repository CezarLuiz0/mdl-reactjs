# Material Design Lite for React JS
This project is under development and will be breaking.

**Do not use in production!**

[You can find documentation and examples here](https://zarxor.github.io/mdl-reactjs/docs/)

## Usage ##
To install run
``` npm install mdl-reactjs ```
Then include by using
``` var MDL = require('mdl-reactjs') ```


Current components:
``` <MDL.Badge /> ```
``` <MDL.Button /> ```
``` 
<MDL.Card>
	<MDL.CardTitle />
	<MDL.CardSupportingText />
	<MDL.CardActions />
	<MDL.CardMenu />
</MDL.Card>
```
``` 
<MDL.Layout>
	<MDL.LayoutHeader>
		<MDL.LayoutHeaderRow />
	</MDL.LayoutHeader>
	<MDL.LayoutDrawer>
		<MDL.LayoutTitle />
	</MDL.LayoutDrawer>
	<MDL.LayoutContent />
	<MDL.CardMenu />
</MDL.Layout>
```
